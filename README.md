# Gyroflow 批量处理工具 🎥

基于 /Volumes/xiaomi13/gyzidonghua/ 的最新版本。

双击 `Gyroflow批量处理.app` 使用，或直接运行：
```
python3 gyroflow_batch_gui.py
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `gyroflow_batch_gui.py` | GUI 版主程序（原生 macOS 对话框） |
| `gyroflow_batch.py` | CLI 简化版 |
| `gyroflow_batch.py使用教程.txt` | 使用教程 |
| `Gyroflow批量自动化处理流程.md` | 处理流程说明 |
| `grrofolw-json/` | Gyroflow JSON 配置样本 |

## 依赖

- Gyroflow.app（`/Applications`）
- Python 3.9+
- pyautogui（GUI 版需要）
